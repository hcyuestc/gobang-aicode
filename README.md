# 五子棋游戏

一个使用HTML、CSS和JavaScript开发的五子棋游戏，支持响应式设计，可在桌面和移动设备上运行。

## 游戏特性

- 完整的五子棋游戏规则实现
- 响应式设计，适配不同屏幕尺寸
- 支持桌面端（鼠标点击）和移动端（触摸操作）
- 游戏状态显示和分数记录
- 悔棋功能
- 重新开始游戏功能
- 美观的用户界面和动画效果

## 如何开始游戏

1. 打开 `index.html` 文件在浏览器中运行游戏
2. 黑方先行，点击棋盘交叉点落子
3. 黑白双方交替落子
4. 率先在横、竖、斜方向上连成五子者获胜
5. 使用控制面板上的按钮可以悔棋或重新开始游戏

## 技术实现

- **HTML**: 构建游戏界面结构
- **CSS**: 实现游戏界面样式，包括响应式设计和动画效果
- **JavaScript**: 实现游戏逻辑，包括棋盘管理、落子、胜负判断等功能

## 单元测试

项目内置了一些针对核心胜负逻辑的 Jest 单元测试，运行步骤如下：

1. 安装依赖：`npm install`
2. 运行测试：`npm test`

## 游戏规则

1. 游戏在15×15的棋盘上进行
2. 黑方先行，黑白双方交替落子
3. 只能在棋盘的交叉点上落子
4. 率先在横、竖、斜方向上连成五子者获胜
5. 如果棋盘已满且无人获胜，则为平局

## 开发说明

本游戏使用纯前端技术实现，无需后端支持。游戏数据存储在浏览器内存中，刷新页面会重置游戏状态。
